Detecting & Preventing Phishing Using Machine Learning

A group project focused on using machine learning techniques to classify URLs as phishing or benign. We worked with Python, data preprocessing, and models from scikit-learn using real phishing datasets.

The project explores feature extraction from URLs, model evaluation, and how this type of tool could support security teams in detecting phishing attempts faster.

National Cyber League Challenges

Through NCL, I’ve worked on challenges involving OSINT, password cracking, log analysis, network traffic inspection, and web-based vulnerabilities — the same kinds of skills used by red and blue teams.

These challenges strengthened my ability to think like an attacker while staying grounded in documenting steps, explaining findings, and approaching problems methodically.
